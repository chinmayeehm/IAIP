# Voice Assistant using Python

This project is a Python-based voice assistant developed as part of the Intern Alpha Python Programming Internship.

## Features
- Voice input using SpeechRecognition
- Voice output using pyttsx3
- Responds to greetings
- Tells current time
- Keyword-based command handling
- Exit command support

## Technologies Used
- Python
- SpeechRecognition
- pyttsx3

## How to Run
1. Install Python
2. Install dependencies:
   pip install -r requirements.txt
3. Run the program:
   python main.py

## Sample Commands
- "Good evening"
- "Hello"
- "What is the time"
- "Help"
- "Exit"
